# SEO Techniques :
Semantic HTML: used HTML5 semantic elements (e.g., <header>, <nav>, <section>, <footer>) to clearly define the structure and meaning of content for search engines.
Meta Descriptions: used <meta name="description"> tags that summarize the page's content, added relevant keywords.
Header Tags (H1-H6): Used header tags to create a clear content hierarchy.
Image Optimization: Used descriptive alt attributes for all images.
